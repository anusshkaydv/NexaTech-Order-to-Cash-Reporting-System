Technology Stack Used

Python Development

Python scripts generate all master datasets and transactional datasets. The ETL framework performs validation, transformation, logging, reporting and uploading automatically.

Azure Cloud Platform

Business datasets are uploaded into Azure Blob Storage before being orchestrated through Azure Data Factory and stored inside Azure SQL Database.

SQL Data Warehouse

The ERP schema consists of sixteen normalized business tables with relationships, constraints and dedicated Analytics SQL Views for reporting.

Business Intelligence

Power BI consumes the Analytics Views to create executive dashboards for Sales, Customers, Inventory, Finance, Orders, Returns and Products.

Project Workflow

Data Generation

Python Generator Scripts automatically create realistic ERP datasets including customers, products, orders, payments, inventory, invoices, shipments and returns.

Cloud Integration

Generated datasets are uploaded into Azure Blob Storage where Azure Data Factory manages movement into Azure SQL Database.

Analytics Layer

Business logic is implemented through SQL Views to calculate KPIs required by management reporting.

Reporting Layer

Power BI connects to Azure SQL Analytics Views to provide interactive business dashboards.
